# `@slum-ui/input`

Slum ui input component.

## `install`

```bash
npm install @slum-ui/input
```

## `demo`

Take a look at out public Storybook that has all the available components with their various configuration editable:

https://slumlords.london/slum-ui//

## `usage`

### `example`
```html
<slum-input
  variant="cyan"
  size="medium"
  state="success"
></slum-input>
```
