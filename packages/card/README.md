# `@slum-ui/card`

Slum ui card component.

## `install`

```bash
npm install @slum-ui/card
```

## `demo`

Take a look at out public Storybook that has all the available components with their various configuration editable:

https://storybook.slumlord.london/

## `usage`

### `example`
```html
<slum-card
  variant="cyan"
></slum-card>
```
